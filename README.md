# Excel Sales Dashboard – Data Analyst Internship (Task 1)

## Project Overview

This project demonstrates the use of Microsoft Excel to clean, analyze, and visualize sales data.
The goal was to transform raw sales data into meaningful insights using Pivot Tables, charts, and an interactive dashboard.

## Objective

* Clean and analyze the provided dataset (Superstore Sales Data)
* Build Pivot Tables to summarize key metrics
* Create charts to visualize sales insights
* Develop a dashboard to present the analysis clearly

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Conditional Formatting

## Key Insights Visualized

* **Total Sales Analysis**
* **Profit by Category**
* **Sales Trends Over Time**
* Identification of **high and low performing categories** using conditional formatting

## Project Files

* **task1.xlsx** – Excel workbook containing the dataset, pivot tables, and dashboard
* **pivot table.png** – Preview of pivot table analysis
* **dashboard.png** – Final interactive dashboard visualization
